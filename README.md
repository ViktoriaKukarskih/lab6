# Лабораторная работа №6. Уведомления
- _Выполнила:_ Кукарских Виктория
- _Язык программирования:_ Java

# Что делает приложение?
- **состоит из 3 экранов:**
  - [Главный экран](#activity1)
  - [Экран добавления напоминания](#addReminder)
  - [Экран с детальным содержанием по каждому напоминанию](#detail)

- позволяет:
  - _устанавливать_ напоминания на определенное время и_ отправлять_ уведомления устройству
  - по тапу на уведомление в панели _открывать_ детальную карточку напоминания
  - по лонгтапу на напоминание на главном экране _удалять_ его

# <a id="activity1">Главный экран</a>
**Состоит из:**
- _списка установленных напоминаний:_
    - составляющие списка:
       - заголовок напоминания
       - дата, когда отправится уведомление
   - действия со списком:
       - удалить напоминание можно по лонгтапу на него
       - просмотреть детальную информацию можно по тапу на него
- _кнопки "Добавить напоминание"_. По тапу открывается [Экран добавления напоминания](#addReminder)


# <a id="addReminder">Экран добавления напоминания</a>
**Состоит из:**
- _2х полей ввода:_
   - заголовка напоминания
   - текста напоминания
- _3х кнопок:_
   - выбора даты (используя `dataPicker`)
   - выбора времени (используя `timePicker`)
   - сохранения напоминания

# <a id="detail">Экран с детальным содержанием</a>
**Состоит из:**
- _Заголовка напоминания_
- _Полного текста напоминания_
